# VIN后台管理系统

这是一个在学习`vue-cli` `vue-router`时的练手项目，是仿照一个教程来编写的项目。

使用了`element-ui`来作为ui框架。不得不说有框架，开发效率确实非常高，很容易就可以搭建出美观的页面。
